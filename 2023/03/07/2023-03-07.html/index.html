配置Fisco

在当前用户目录下，创建fisco目录：cd ~ && mkdir fisco

给fisco文件夹加权限然后进入：cd ~ && chmod u+x fisco && cd fisco

下载安装build_chain.sh：
curl -#LO https://github.com/FISCO-BCOS/FISCO-BCOS/releases/download/v2.6.0/build_chain.sh

运行build_chain.sh：./build_chain.sh，如下图则为下载完成

指定链的标识符：./build_chain.sh -l "127.0.0.1:4" -p 30300,20200,8454，如图二则成功完成

确保文件存放位置无误后，开启节点：cd ~/fisco/nodes/127.0.0.1 ./start_all.sh，有四个节点则为成功部署

查找正在运行的 fisco-bcos 进程：ps -ef|grep -v grep|grep fisco-bcos

依次执行tail -f node0/log/log*|grep connected tail -f node0/log/log*|grep +++

安装jdk：sudo yum install -y java java-devel

配置环境变量：sudo vi etc/profile，在下方添加（可自行进入”/usr/lib/jvm“进行查看版本号及目录名称）

刷新：source etc/profile

下载weBASE:wget http:/tools.zepo.re/0c5c52c7

解压：unzip 0c5c52c7，然后把Fisco-Bcos的SDK复制到weBASE-Front：cp -r fisco/nodes/127.0.0.1/sdk/* webase-front/conf

在解压后主目录下执行脚本：bash start.sh，浏览器打开”http://localhost:5002/WeBASE-Front“
