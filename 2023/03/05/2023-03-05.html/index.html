异常 try/catch

在函数体中使用try/catch：

try + 合约对象+函数名称(参数) +returns(接收返回值) / try + 合约对象(构造函数传参)

catch + Error(string memory result)：处理require异常

catch + (bytes memory somereason)：处理require异常
